Dependências

Para executar o programa em si, não há nenhuma dependência, porém, para criar as imagens geradas pelo GraphViz, é necessário a instalação do pacote do GraphViz. No Ubuntu 20.04:

$ sudo apt install graphviz

Execução

Para compilar o programa:

$ make

Para rodar o programa:

$ ./main <NOME_DO_ARQUIVO>

Para gerar os grafos em png e svg, que ficarão no diretório dot/:

$ make dot
